# Machine Learning-Based CSI Feedback With Variable Length in FDD Massive MIMO

This repository contains code to reproduce the paper:

> M. Nerini, V. Rizzello, M. Joham, W. Utschick, B. Clerckx,
> "[Machine learning-based CSI feedback with variable length in FDD massive MIMO](https://ieeexplore.ieee.org/document/9928062),"
> IEEE Trans. Wireless Commun., 2022.

## Code

The dataset can be downloaded [here](https://www.dropbox.com/scl/fo/qrwfwygxs801qoh88miwd/h?dl=0&rlkey=xblkn11p83eu0mn2j8hcksjtp).

The script `main.m` generates the results needed to reproduce Fig. 4 in the paper.
These results are saved in the folder `results` to ease the reproduction.
